typeKeys

The typeKeys action types keys into a focused element, supporting special key combinations.

Description

Type a string of keys into a focused element. Supports special keys like Enter, Tab, and modifier combinations.

Fields

The following fields are available for the typeKeys action:

  • keys - The keys to type.
  • selector - The element to type into.
  • delay - Delay between key presses in milliseconds.

Examples

Here are some examples of typeKeys usage:

{ "action": "typeKeys", "keys": "Hello World" }
